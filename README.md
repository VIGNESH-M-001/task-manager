# Task Manager

A simple Task Manager application built using HTML, CSS, JavaScript, Node.js, Express.js, and SQLite.

## Features

- Add new tasks
- View tasks
- Mark tasks as completed
- Delete tasks
- Store task data using SQLite

## Technologies Used

- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- SQLite

## Installation

1.Clone the repository:
git clone https://github.com/VIGNESH-M-001/task-manager.git

2.Navigate to the project directory:
cd task-manager

3.Install backend dependencies:
cd backend
npm install

4.Start the server:
node index.js

5.Open the frontend in your browser.
